# Code of Conduct

## Our pledge

We want this project to be a welcoming place for everyone learning about or
hacking on the LinkStar H68K, regardless of experience level or background.

## Standards

**Do:** be respectful, assume good faith, help newcomers, keep feedback about the
work, and credit others' findings.

**Don't:** harass, demean, or troll; post others' private information; or derail
issues with off-topic disputes.

## Scope

Applies to all project spaces — issues, pull requests, discussions — and to public
spaces when representing the project.

## Enforcement

Maintainers may edit, remove, or reject contributions and comments that violate
this code, and may block repeat offenders. Report problems by opening a
confidential issue or contacting the maintainer.

This is a lightweight adaptation of the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/). Be kind.
